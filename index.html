Ce site Web donne accès à la documentation et aux codes de différentes librairies Javascripts (accessible sur http://mdst-macroes.ird.fr/cdn). Ces librairies permettent l’intégration rapide de services développés à l'origine pour des sites Web liés à l'unité MARBEC (http://www.umr-marbec.fr) afin de les réutiliser dans d'autres Sites Web (IRD, partenaires ou autres):

  • prestation 1: la librairie Javascript "Atlas Fact Sheet" permet de mettre en place rapidement des fiches sur des objets de référence (Fiches sur des agents, des bases de données ou des espèces) à partir d'information stockées dans des bases de données accessibles en ligne (Wiki/Dbpédia ou Ecoscope pour les informations propres à l’unité MARBEC).
  • prestation 2: application de la méthode pour la mise en place de la librairie Javascript "MDST data access" permet d’intégrer un service de requêtes sur des bases de données connectées au MDST
  • prestation 3: la librairie Javascript "Map visualization" permet d’intégrer un service de visualisation de données grillées (pêche, satellite, sorties de modèles: exemple http://mdst-macroes.ird.fr/MDSTWebClientNG/SIP/SIP.action?selectedSourceType=SATELLITE&selectedProcessURI=sst2)
  • prestation 4: la librairie Javascript "Indicators" permet d’intégrer un service de requêtes sur des bases de données connectées au MDST (Tuna Atlas / Sardara, séries temporelles de paramètres physico-chimiques, profils de profondeur:http://mdst-macroes.ird.fr/MDSTWebClientNG/SIP/SIP.action?selectedSourceURI=http://www.ecoscope.org/ontologies/resources/dbSardaraTunaAtlas&selectedViewURI=http://www.ecoscope.org/ontologies/resources/dbSardaraTunaAtlasI1I2&selectedProcessURI=org.n52.wps.server.r.toto_wps)
Ce site montre:
  • comment créer rapidement une nouvelle page Web qui intègre ces services,
  • comment les intégrer dans des pages existantes.

l'initialisation se fait pas l'instanciation d'un object ayant pour object :

  • URI
  • endpointLocation
